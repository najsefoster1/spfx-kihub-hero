# SPFx KIHub Hero Section

This project is a SharePoint Framework (SPFx) solution that enhances the visual experience of SharePoint pages by creating a modern, customizable hero section.

## Overview

The KIHub Hero Section was designed to improve the first impression and usability of SharePoint pages by replacing standard layouts with a more visually engaging and structured header area.

It allows for better highlighting of key content, tools, and messaging while maintaining a clean and modern design.

## Key Features

- Custom hero section for SharePoint modern pages  
- Improved visual hierarchy and layout  
- Supports custom backgrounds and content styling  
- Enhances overall user experience and page engagement  

## Business Value

This solution helps create a more polished and intentional SharePoint experience by improving how information is presented at the top of pages. It supports better communication, user engagement, and adoption of key tools and resources.

## Tech Stack

- SharePoint Framework (SPFx)  
- TypeScript  
- React  

## Getting Started

npm install  
gulp serve  

## Build and Package

gulp bundle --ship  
gulp package-solution --ship  

## Deployment

1. Upload the .sppkg file to the SharePoint App Catalog  
2. Deploy the solution  
3. Add the solution to your target SharePoint site
